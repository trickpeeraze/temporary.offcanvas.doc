Temporary.offcanvas

push • pull • panel • practice

Temporary.offcanvas is a jQuery plugin for setting up the popular feature |offcanvas| to your website with ease. Custom the plugin and use on your site please see the instruction below.

Getting Started

This jQuery Plugin require jQuery 1.10+ Library and jquery.transit plugin of @ricostacruz
At my first state, I need help from other libraries to make a job done and running smooth. Using of jquery.transit is good and then can also be used its cool feature on your site.


first you have to download the source code from here or my Github


Now add this to the head of your document:

	<script src="jquery.min.js" ></script>
	<script src="jquery.transit.min.js" ></script>
	<script src="temporary.offcanvas.js" ></script>
	
	<!-- Optional -->
	<script src="jquery.easing.js" ></script>

and now put the script:
the canvas will be out from an element you bind with the main jQuery object.

canvas = $('.container').offcanvas({
	// size is a required option
	size: '250'
	// and add some content to the canvas
	content:'some content in the canvas',
});

or with these options:
see more information all option and description.

canvas = $('.container').offcanvas({
	// all option and its default value
	content: '',
	size: 0,
	position: 'right',
	canvasClass: 'temp-canvas-wrapper',
	canvasPadding: '15px',
	offset: 0,
	duration: 220,
	easing: 'linear',
	toggleButtonSelector: '',
	pushAndPull: false,
	tapToClose: false,
	swipeToClose: false,
	onBeforeOpen: function(){},
	onOpen: function(){},
	onBeforeClose: function(){},
	onClose: function(){}
});

then use offCanvas API method to manage your canvas:
see more information all API methods and description.

$('#some').on('click', function(){
	canvas.open();
	return false;
});

That's it!
see more at the examples section

Examples

Toggle Button

Using [toggleButtonSelector] option to automatic bind open/close button (toggle) without extra coding by putting a jQuery button element through the option value

toggle 
 toggleButtonSelector: $('some button')

or you can write your own toggle function by using the API method: this is simply example one

$('some button').on('click', function(){
	// cv is canvas object
	if(cv.isClose){
		cv.open();
	} else {
		cv.close();
	}
	return false;
});

Position

Using [position] to specific your position of the canvas. It can be possible with 'left', 'right' (default), 'top', and 'bottom'.

top right bottom left 
	position: 'top'

Push Pull and Tab

Using [pushAndPull] set it to true when you need the whole area be put agaist the direction of the canvas position. and [tapToClose] the canvas can be closed with click anywhere else on the page.

push

<mark-up>

<div id="example-push">
	<div class="content">
		<div></div>
		<div></div>
		...
		<div></div>
	</div>
</div>

<script>

exp1 = $('#example-push').offcanvas({
	content: 'content ...',
	size: '25%',
	position: 'left',
	pushAndPull: true,
	tapToClose: true
});

Usage

Call the plugin by using .offcanvas() add parameter as an object of option. offcanvas object will be returned then you can use offcanvas API methods.

canvasObject = $('viewport selector').offcanvas();

Options

Name Type Default Description
Config
content Text | HTML | $(Object) empty

The content inside the canvas it can be text, HTML, or even jQuery object

note: if the jQuery object is input that element will be moved into the canvas autometically
size pixel unit 0

Size of the canves represented as heigth if position is set to 'top' or 'bottom' and as width if it is set to 'left' or 'right'.

note: the unit can be set in percentage.
position top | right | bottom | left right

specific where is the position of the canvas while it is showing.
canvasClass string temp-canvas-wrapper

css class specific in the canvas wrapper.
canvasPadding padding unit 15px

padding inside between canvas and content.
offset pixel unit 0

that is when you need to set some distance from the edge of the parent direction is related to the position

duration Number 220

period of time of canvas slide animation.
easing Easing Type linear

easing slide effect. You can use jquery.easing to extend more ease effects
toggleButtonSelector string empty

an query string of selector that represented as button to click open and close the canvas automatically in easiest way.
fixedPosition Boolean true

in case that your canvas is not fixed on the page but within an element this option is necessary to set to false.

note: if it is set to false within the element that element cannot to scroll (because of effect of overflow:hidden).
pushAndPull Boolean false

option to set it whether when the canvas is opened the whole area will be pushed by the canvas instead of floating over.
tapToClose Boolean false

option to set it whether when the user clicks anywhere but canvas, the canvas will be closed auto.
swipeToClose Boolean false

option that support for mobile when user need to close the canvas just swipe agianst to the position of the canvas.
Events
onBeforeOpen function() empty

fire an event function while the canvas is opening.
onOpen function() empty

fire an event function when the canvas is opened.
onBeforeClose function() empty

fire an event function when the canvas is closing.
onClose function() empty

fire an event function when the canvas is closed.

Method & API

Name Type Return Description
open() Method void

a function that triggers the canvas to open.
close() Method void

a function that triggers the canvas to close.
isOpen API Boolean

an API that check whether the canvas has being opened or not.
isClose API Boolean

an API that check whether the canvas has being closed or not.
destroy Method void

a function that destroy the canvas object.
